# Rafael Henrique

**`Desenvolvedor Back-end`**

Me chamo Rafael Henrique, tenho 20 anos e estudo desenvolvimento back-end de forma autodidata, com foco na criação de APIs, aplicações utilitárias e projetos práticos.

---

### Projetos em destaque

**CleanTube – Stay Focused**
Extensão de navegador publicada, desenvolvida para reduzir distrações ao ocultar conteúdos curtos em plataformas como YouTube, Instagram e TikTok.

**ContratEasy**
API back-end para geração e gerenciamento de contratos personalizados, com autenticação de usuários, controle de acesso e integração com banco de dados.

**Minecraft Tick Converter**
Aplicação web publicada para conversão de tempo em ticks do Minecraft, com interface própria, sistema de temas e versão desktop com Electron.
